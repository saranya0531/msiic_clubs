The Event Management Web Application is a Flutter-based website designed to manage and streamline event coordination processes.

The platform provides real-time event updates, automated registration, and notification features, serving 300+ active users efficiently.

This system significantly reduced manual event coordination efforts by 50% through process automation.

🎯 Objectives

Digitize event management processes

Enable real-time event updates

Automate registration and notifications

Improve user engagement and accessibility

🚀 Key Features

📢 Real-time event announcements

📝 Online event registration system

🔔 Automated email / in-app notifications

👥 User-friendly dashboard

📊 Admin panel for event management

📱 Responsive web design

📊 Impact & Achievements
